# resourcesの利用方法

## 共通事項

- 自社開発のCSSやJSは、appに保存する。サブディレクトリは適度に作成する。
- AdminLTEは、resourcesの直下に保存する。
- AdminLTE/pluginsに入っていないプラグインを追加する場合は、resources/pluginsに保存する。
